# distance(value, fixed, unit) ⇒ string

格式化距离字段带单位输出

小于1000时单位为'm'，大于1000时单位为'km'

Kind: global function

Param Type Description
value Number 要转换的距离数值（单位：米）
fixed Number 保留小数位数
unit Object 单位名称，默认（{ m: 'm', km: 'km' }）

Example

distance(2520);
// => '2.5km'

distance(520);
// => '520m'

# measureDistance(fromPoint, toPoint) ⇒ Number

测算两个经纬度之间的物理距离

Kind: global function
Returns: Number - 距离值（单位：米）

Param Type Description
fromPoint Object 坐标地址1
toPoint Object 坐标地址2
