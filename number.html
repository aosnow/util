# Functions

isNumber(value) ⇒ boolean

检查 value 是否是原始 Number 数值型 或者 数值常量对象

isNumeric(value) ⇒ boolean

检查 value 是否为全数字内容

注：该方法包含所有 isNumber(value) 为 true 的情况，以及字符串类型的数字。即该方法比 isNumber 的条件更宽泛，可用于检测任意数字内容出现的场景。

注：NaN 也属于一个合法的数值型。

isNaN(value) ⇒ boolean

检查 value 是否是 NaN。

注意: 这个方法基于 Number.isNaN，和全局的 isNaN 不同之处在于，全局的 isNaN 对于 undefined 和其他非数字的值返回 true。

random([lower], [upper], [floating]) ⇒ number

产生一个包括 lower 与 upper 之间的随机数。 如果只提供一个参数返回一个0到提供数之间的数。 如果 floating 设为 true，或者 lower 或 upper 是浮点数，结果返回浮点数。

注意: JavaScript 遵循 IEEE-754 标准处理无法预料的浮点数结果。

toFixed(value, size, [processor]) ⇒ Number

保留指定位数的小数

# isNumber(value) ⇒ boolean

检查 value 是否是原始 Number 数值型 或者 数值常量对象

Kind: global function

Param Type Description
value * 要检查的值。

Example

isNumber(3);
// => true

isNumber(Number.MIN_VALUE);
// => true

isNumber(Infinity);
// => true

isNumber('3');
// => false

# isNumeric(value) ⇒ boolean

检查 value 是否为全数字内容

注：该方法包含所有 isNumber(value) 为 true 的情况，以及字符串类型的数字。即该方法比 isNumber 的条件更宽泛，可用于检测任意数字内容出现的场景。

注：NaN 也属于一个合法的数值型。

Kind: global function

Param Type Description
value * 要检查的值。

Example

isNumeric(Number.MIN_VALUE);
// => true

isNumeric(true);
// => false

isNumeric(3);
// => true

isNumeric('3');
// => true

# isNaN(value) ⇒ boolean

检查 value 是否是 NaN。

注意: 这个方法基于 Number.isNaN (opens new window)，和全局的 isNaN (opens new window) 不同之处在于，全局的 isNaN (opens new window) 对于 undefined 和其他非数字的值返回 true。

Kind: global function
Returns: boolean - 如果 value 是一个 NaN，那么返回 true，否则返回 false。

Param Type Description
value * 要检查的值。

Example

isNaN(NaN);
// => true

isNaN(new Number(NaN));
// => true

isNaN(undefined);
// => false

# random([lower], [upper], [floating]) ⇒ number

产生一个包括 lower 与 upper 之间的随机数。 如果只提供一个参数返回一个0到提供数之间的数。 如果 floating 设为 true，或者 lower 或 upper 是浮点数，结果返回浮点数。

注意: JavaScript 遵循 IEEE-754 标准处理无法预料的浮点数结果。

Kind: global function
Returns: number - 返回随机数。

Param Type Default Description
[lower] number 0 下限。
[upper] number 1 上限。
[floating] boolean 指定是否返回浮点数。

Example

random(0, 5);
// => 0, 1, 2, 3, 4, 5

random(5);
// => 0, 1, 2, 3, 4, 5

random(5, true);
// => 0 到 5 之间的浮点数（如：1.6288619395308945，0.5192344462779486...）

// lower 或 upper 为小数，且 floating 为 false，则严格按照对应的小数位数返回随机数结果
random(1, 1.05);
// => 1, 1.01, 1.02, 1.03, 1.04, 1.05

# toFixed(value, size, [processor]) ⇒ Number

保留指定位数的小数

Kind: global function

Param Type Default Description
value Number 原数值
size Number 需要保留的小数位数
[processor] function Math.round 截断小数的处理函数，默认使用四舍五入函数

Example

toFixed(1.468468, 2);
// => 1.47

toFixed(1.468468, 2, Math.floor);
// => 1.46
