# isBoolean(value) ⇒ boolean

检查 value 是否是原始 Boolean 型

Kind: global function

Param Type Description
value * 要检查的值。

Example

isBoolean(true);
// => true

isBoolean(false);
// => true

isBoolean(0);
// => false

# isFalse(value) ⇒ boolean

检查 value 是否是 Javascript 认为的 false 值（包括 null,undefinded,false,NaN,空字符串，空数组）

Kind: global function

Param Type Description
value * 要检查的值。

Example

isFalse(true);
// => false

isFalse(false);
// => true

isFalse('');
// => true

isFalse([]);
// => true

isFalse(0);
// => false
